1. RNN 结构和内部计算

RNN结构示意图（这里没画bias， 但公式里写了，借的李宏毅教授PPT图）

xxx为当前状态下数据的输入， hhh表示接收到的上一个节点的输入。

yyy为当前节点状态下的输出，而 $h^{\prime} $为传递到下一个节点的输出。

真实计算公式为：

\begin{align} \boldsymbol{h}_t &= \text{tanh} (\boldsymbol{W}_h [\boldsymbol{x}_t, \boldsymbol{h}_{t-1}] + b_t) \\ \\ \boldsymbol{y}_t &= \text{tanh} (\boldsymbol{W}_0 \boldsymbol{h}_t + b_t) \end{align}

通过上图的公式可以看到，输出h′\boldsymbol{h}^{\prime}h′与 x\boldsymbol{x}x 和 h\boldsymbol{h}h的值都相关。

而 y\boldsymbol{y}y 则常常使用 h′\boldsymbol{h}^{\prime}h′投入到一个线性层（主要是进行维度映射）然后使用softmax进行分类得到需要的数据。

2. LSTM

LSTM是long short-term memory,与RNN区别是

  • LSTM两个传递状态：$c^t $ cell state, hth^tht hidden state
  • RNN一个传递状态: hth^tht

注： RNN中hth^tht等价于 LSTM中 ctc^tct

而对于input gate, forget gate, output gate中是[xt,ht−1][x^t, h^{t-1}][xt,ht−1]向量拼接后乘以相应权重矩阵再过sigmoid函数转换为0到1的数值，作为门控状态。

需要注意的是z是tanh函数，转换为-1到1，因为是输入数据不是门控状态。

因此有下面4个公式：（忽略bias）

z=tanh(W[xt,ht−1])zi=σ(Wi[xt,ht−1])zf=σ(Wf[xt,ht−1])zo=σ(Wo[xt,ht−1])\begin{aligned} \boldsymbol{z} &= \text{tanh}(\boldsymbol{W} [\boldsymbol{x}^t, \boldsymbol{h}^{t-1}]) \\ \boldsymbol{z}^i &= \sigma(\boldsymbol{W}^i [\boldsymbol{x}^t, \boldsymbol{h}^{t-1}]) \\ \boldsymbol{z}^f &= \sigma(\boldsymbol{W}^f [\boldsymbol{x}^t, \boldsymbol{h}^{t-1}]) \\ \boldsymbol{z}^o &= \sigma(\boldsymbol{W}^o [\boldsymbol{x}^t, \boldsymbol{h}^{t-1}]) \end{aligned} zzizfzo​=tanh(W[xt,ht−1])=σ(Wi[xt,ht−1])=σ(Wf[xt,ht−1])=σ(Wo[xt,ht−1])​

注：[]表示两个向量拼接，而 W\boldsymbol{W}W四个权重矩阵都是训练学习得到的。

然后根据上面四个信号$ \boldsymbol{z}, \boldsymbol{z}^f, \boldsymbol{z}^i, \boldsymbol{z}^o $, 由下列公式得到输出， cell state和隐藏状态。

ct=zf⊙ct−1+zi⊙zht=zo⊙tanh ctyt=σ(W′ht)\begin{aligned} \boldsymbol{c}^t &= \boldsymbol{z}^f \odot \boldsymbol{c}^{t-1} + \boldsymbol{z}^i \odot \boldsymbol{z} \\ \boldsymbol{h}^t &= \boldsymbol{z}^o \odot \text{tanh }\boldsymbol{c}^{t} \\\boldsymbol{y}^t &= \sigma (\boldsymbol{W}^\prime\boldsymbol{h}^{t}) \end{aligned} cthtyt​=zf⊙ct−1+zi⊙z=zo⊙tanh ct=σ(W′ht)​

  • cell state ：等于 上一个cell state 与遗忘门信号 $\boldsymbol{z}^f $ Hadamard product，和 输入门信号 $ \boldsymbol{z}^i 与输入信号与输入信号与输入信号\boldsymbol{z}$ 的Hadamard product之和
  • hidden state: 等于输出门信号 $\boldsymbol{z}^o $ 与 过tanh后的 cell state 的Hadamard product
  • 输出： 等于上一个时间步后权重矩阵W\boldsymbol{W}W 与 隐藏状态过sigmoid

3. GRU

GRU(Gate Recurrent Unit)结构示意图如下：

最重要的是，两个门控reset gate 和 update gate:

rt=σ(Wr[xt,ht−1])zt=σ(Wz[xt,ht−1])\begin{aligned} \boldsymbol{r}^t &= \sigma(\boldsymbol{W}^r [\boldsymbol{x}^t, \boldsymbol{h}^{t-1}]) \\ \boldsymbol{z}^t &= \sigma(\boldsymbol{W}^z [\boldsymbol{x}^t, \boldsymbol{h}^{t-1}]) \end{aligned} rtzt​=σ(Wr[xt,ht−1])=σ(Wz[xt,ht−1])​

然后得到重置的隐藏层状态${\boldsymbol{h}{t-1}}\prime= \boldsymbol{h}^{t-1} \odot\boldsymbol{r}^{t} .将这个隐藏状态与. 将这个隐藏状态与.将这个隐藏状态与\boldsymbol{x}^t$ 拼接再过tanh将重置的隐藏状态缩放为-1~1.得到ht′{\boldsymbol{h}^{t}}^\primeht′

即

ht′=tanh (W[xt,ht−1′])=tanh (W[xt,ht−1⊙rt){\boldsymbol{h}^{t}}^\prime= \text{tanh }(\boldsymbol{W}[\boldsymbol{x}^t, {\boldsymbol{h}^{t-1}}^\prime])=\text{tanh }(\boldsymbol{W}[\boldsymbol{x}^t, \boldsymbol{h}^{t-1} \odot\boldsymbol{r}^{t}) ht′=tanh (W[xt,ht−1′])=tanh (W[xt,ht−1⊙rt)

这里ht′{\boldsymbol{h}^{t}}^\primeht′ 主要包含了当前输入$ {\boldsymbol{x}^{t}} $数据, 还有有选择地添加上一个时间隐藏状态，这里主要是重置门信号作用(你可以理解为乘以一个0~1的数来控制添加隐藏状态的多少)。

最后一步是ht\boldsymbol{h}^{t}ht，也是输出。计算公式为：

yt=ht=(1−zt)⊙ht−1+zt⊙ht′\boldsymbol{y}^{t} =\boldsymbol{h}^{t} = (1-\boldsymbol{z}^t ) \odot \boldsymbol{h}^{t-1} + \boldsymbol{z}^t \odot {\boldsymbol{h}^{t}}^\prime yt=ht=(1−zt)⊙ht−1+zt⊙ht′

  • ht′{\boldsymbol{h}^{t}}^\primeht′ 相当于LSTM中的hidden state : $ \boldsymbol{h}^t = \boldsymbol{z}^o \odot \text{tanh }\boldsymbol{c}^{t} $

  • 而$\boldsymbol{h}^{t} $ 相当于cell state

实际上，GRU是用一个update gate来替换LSTM的input gate 和 forget gate.

在 LSTM 中这两个门分别是

  • 控制输入多少(输入是上一个时间状态和当前输入总的信息量)
  • 控制遗忘信息量的多少

而update gate就是上面的zt\boldsymbol{z}^tzt ,是一个0~1的控制信号。那么：

  • $\boldsymbol{z}^t $理解等价于input gate 控制信号， zt⊙ht′\boldsymbol{z}^t \odot {\boldsymbol{h}^{t}}^\primezt⊙ht′ 就等效于经过input gate后的信息
  • (1−zt)(1-\boldsymbol{z}^t )(1−zt) 等价于 forget gate 控制信号， (1−zt)⊙ht−1(1-\boldsymbol{z}^t ) \odot \boldsymbol{h}^{t-1}(1−zt)⊙ht−1 等效于 经过 forget gate后的信息
